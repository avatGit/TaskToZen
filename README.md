# TaskToZen 🧠

[![Translate to French](https://shields.io)](https://github.com/avatGit/TaskToZen)

TaskToZen est une application web de gestion intelligente des tâches conçue pour aider les étudiants, développeurs et personnes multiprojets à garder le cap sur leurs objectifs sans se sentir dépassés.

Le projet est né d’un besoin personnel : organiser efficacement plusieurs activités (projets, apprentissage, objectifs, révisions, etc.) tout en réduisant la surcharge mentale.

---

## Vision du projet

TaskToZen ne vise pas à être une simple to-do list.

L’objectif est de construire un système capable de :

- prioriser intelligemment les tâches
- guider l’utilisateur au quotidien
- réduire le chaos organisationnel
- transformer les objectifs en actions concrètes
- améliorer la concentration et la productivité
